# Customer Retention & Revenue Analysis

## Problem
Analyze customer behavior and revenue trends to understand business performance.

## Tools
- SQL
- Power BI

## Analysis Performed
- Monthly revenue analysis
- Top customers identification
- Repeat customer analysis
- Average Order Value (AOV)

## Key Insights
- Most customers are one-time buyers
- Repeat rate is significantly low
- Revenue depends on new customers

## Recommendations
- Improve customer retention strategies
- Introduce loyalty programs
- Offer discounts for repeat customers
